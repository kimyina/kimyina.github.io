# Yina Kim Personal Website

Static GitHub Pages site for `kimyina.github.io`.

## Files

- `index.html`: homepage
- `styles.css`: site styling
- `projects/`: clickable project detail pages
- `assets/YinaKim_CV.pdf`: public CV without phone number
- `.gitignore`: excludes private source PDFs and the editable CV document

## Deploy

1. Create a GitHub repository named `kimyina.github.io`.
2. Commit these files to the repository root.
3. Push to the `main` branch.
4. In GitHub, open Settings > Pages and use `Deploy from a branch`, `main`, `/root`.

The site URL will be `https://kimyina.github.io`.

Do not publish the original source PDFs, generated working artifacts, or `docx`
file. They are excluded by `.gitignore`; the public CV is
`assets/YinaKim_CV.pdf`.
